# UIEngine
Just a small project that I can use to quickly test small things I think of.

This project was written using *Visual Studio 2022*.

### Requires:
 - A package manager (Im currently using VCPKG).
   https://github.com/microsoft/vcpkg
 - Visual Studio 2022.
